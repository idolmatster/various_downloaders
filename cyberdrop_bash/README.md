# Cyberdrop Bash Downloader

## how to use
./cybash.sh "linktogallery"

## Features

-   Creates Folders with name and id
-   atempts redownload if the there are missing files
-   does not create temporary files
-   only hits the page once

## dependencies

-   pup
-   wget
-   other gnu goodness

## the negatives
- over engeneered
- a bit bloated
- no multidownload
